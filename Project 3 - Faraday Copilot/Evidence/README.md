# Evidence

This folder collects visual and code evidence for the Faraday Copilot portfolio entry.

## Screenshots

- `Screenshots/ui-v0-local-ai-side-panel.png` shows the original local AI side-panel direction.
- `Screenshots/ui-v8-side-panel-tabs.png` shows the later tabbed side-panel UI.
- `Screenshots/ui-v8-fullscreen-workspace.png` shows the fullscreen workspace layout.
- `Screenshots/ui-v8-settings.png` shows settings for API key, model selection, behaviour, writing preferences, and feature flags.

The screenshots are static captures of extension HTML files opened locally. They demonstrate UI structure and visual evolution, but they are not a substitute for running the extension inside Chrome with extension APIs available.

## Code evidence

`Code Evidence/engineering-excerpts.md` summarises the source files that best demonstrate technical skill: Manifest V3 architecture, background messaging, Google Docs companion integration, OpenRouter calls, storage design, and the V8 graph engine.
