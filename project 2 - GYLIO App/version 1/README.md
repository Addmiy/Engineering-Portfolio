# GYLIO MVP Prototype

This workspace contains a no-install front-end prototype for the GYLIO product brief.

## Run

Open `index.html` directly in a browser for the core experience.

For installable PWA behavior and service-worker caching, serve the folder with any static server instead of opening it with `file://`.

## Included MVP Systems

- Morning activation experience with trait-based motivational sequencing
- Structured execution dashboard with schedule awareness and priority tasks
- Responsive visual system designed around calm, disciplined execution
