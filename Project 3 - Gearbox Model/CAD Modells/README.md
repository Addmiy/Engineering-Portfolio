# CAD Evidence

The attached report includes CAD render evidence from Fusion 360, but no original Fusion 360 `.f3d` source files were supplied with the gearbox report.

This folder therefore keeps extracted CAD evidence images from the report:

- `Gearbox CAD Concept Render.png`
- `Gearbox CAD Top View.png`

These images document the intended gear-train layout, support walls, parallel shafts, and planned gear spacing used before fabrication.
